# CA Professional & Financial GST Calculator

A specialized Python-based tool designed for Chartered Accountants to streamline GST compliance and financial calculations with high-speed keyboard shortcuts.

## üöÄ Key Features
- **Memory Logic**: Standard **M+, M-, MR, and MC** buttons to store and sum multiple tax totals.
- **TVM & Finance Ops**: Specialized operators for **x^y (Compounding)**, **1/x (Reciprocals)**, and **Square Roots**.
- **Forward GST**: Calculate 5%, 12%, 18%, and 28% instantly using **F1 - F4** keys.
- **Reverse GST**: Derive base prices from GST-inclusive totals using **Alt + F1 - F4**.
- **Professional Audit Log**: Automatically saves every calculation with a timestamp to `tax_log.csv`.
