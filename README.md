# CA Professional GST Calculator

A specialized Python-based tool designed for Chartered Accountants to streamline GST calculations with high-speed keyboard shortcuts.

## üöÄ Key Features
- **Forward GST**: Calculate 5%, 12%, 18%, and 28% instantly using **F1 - F4** keys.
- **Reverse GST**: Derive base prices from GST-inclusive totals using **Alt + F1 - F4**.
- **Audit Log**: Automatically saves every calculation with a timestamp to `tax_log.csv` for professional record-keeping.
- **Speed Optimized**: Automatically focuses the input box and supports **Numpad Enter** for mouse-free operation.
